complex

complex 可以定义多种数据类型的复数

格式为:complex<数据类型>对象名称（实部值，虚部只）；

eg:
#include
#include
using namespace std;
int main(){
complex comx1(32,34);
complex comx2(32.5,32.6);
cout<<comx1.real()<<” ,”<<comx2.imag<<endl;
return 0;
}

comx1.real为实部,
comx2.imag为虚部；

\t

制表符\t：通常宽度相当于8个空格的位置；

递增运算符和递减运算符

递增递减运算符都有前置（++A）和后置(A++)两种形式
int text=0;
cout << ++text; //此处为前置写法，先计算，再输出；输出为1；

若为 int text=0;
cout << text++; //此处为后置写法，先输出，再计算；输出为0；

逻辑运算

逻辑运算符具有最高优先级
若要判断A是否为偶数：
！A%2 // 这里会先计算！A
//所以正确写法应该是：
！（A%2）；
