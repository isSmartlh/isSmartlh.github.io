2.FILE:
输出文件的 完整路径和文件名；

3.DIR_：
等价于 dirname(_FILE)；
文件所在的目录，若处在被包括的文件中，则返回被包括的文件所在目录；

4.FUNCTION:
所运行的函数名称；

5.CLASS:
类的名称；

6.TRAIT:
代码复用；

7.METHOD:
返回该方法被定义时的名字；

8.NAMESPACE;
当前命名空间的名称；

命名空间

可解决：用户编写的代码与PHP内部的类/函数/常量或第三方类/函数/常量之间的名字冲突；

如果一个文件中包含命名空间，它必须在其他所有代码之前声明；

1.一个文件中可以定义不同的命名空间代码，格式最好如下：
{
    namespace asd{
    }

    namespace aaa{
    }
};

2.将全局的非命名空间中的代码与命名空间中的代码组合在一起,为以下格式：
{
     namespace aaa{
     }

     namespace{//全局变量；
     }
}

3.在声明命名空间之前唯一合法的代码时用于定义源文件的编码方式的eg：declare(encoding='UTF-8')

4.命名空间的名字可以使用分层次的方式定义；
